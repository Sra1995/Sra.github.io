EngimaPy

EnigmaPy is a Python implementation of the legendary Enigma machine used for encryption during World War II. This project allows users to explore the fascinating world of cryptography by simulating the operation of an Enigma machine in a terminal window. Users can set up rotors, configure plugboard pairs, and encrypt/decrypt messages, all while experiencing the thrill of encoding secrets in the spirit of historical cryptography.
