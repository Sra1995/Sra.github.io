File Organization Automation

This project automates the organization of downloaded files using a Python script. The script categorizes files into specific folders based on their types, such as images, music, videos, documents, compressed files, executables, archives, and others. It employs the Watchdog library to monitor changes in the download folder and trigger the sorting process when new files are added.
